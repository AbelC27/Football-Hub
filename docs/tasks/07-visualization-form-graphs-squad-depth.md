# Task: Visualization - Form Graphs and Squad Depth

## Goal
Add visual analytics for team form and depth quality by position.

## Form Graphs
- Last 5 and last 10 match trend.
- Points, goals for, goals against.
- Home vs away split.

## Squad Depth View
- Position stacks (starter quality vs bench quality).
- Injury or suspension availability indicator.
- Rotation risk for upcoming fixtures.

## Data Requirements
- Match results history.
- Player roles and minutes.
- Availability data if accessible.

## UI Requirements
- Clear, compact charts for desktop and mobile.
- Consistent color coding for win/draw/loss and role levels.
- Tooltip details on hover/tap.

## Acceptance Criteria
- Team page shows form graph with selectable windows.
- Squad depth visual updates when squad changes.
- Visualization load time stays acceptable with caching.

## Milestones
1. Define visualization data contracts.
2. Build backend computed metrics.
3. Build chart and depth UI components.
4. Validate usability on mobile.
